# Password_manager
A CLI-based password manager which creates database in your local storage and stores encrypted passwords


## Overview

Password_manager is a secure, command-line-based password management system that facilitates safe storage, retrieval, editing, and deletion of credentials within an encrypted database. Built with a focus on data confidentiality, it employs AES encryption and user authentication to safeguard sensitive information.
